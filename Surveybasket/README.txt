Project Name: Survey Basket
Group members: Andy Peng(apeng), Kechun Mao(kmao)
Description: Survey basket is a survey website for posting, taking and distributing surveys. It has a credit system to encourage people to take surveys.  
How to operate it: run “.mongodb-osx-x86_64-2.2.2/bin/mongod” in Surveybasket folder and run “node server.js”. (We will possibly host it on heroku).
Modules and Libraries we used: 
backend: mongodb, mongoose, express, nodejs, passport
frontend: jQuery, underscore, bootstrap
Link to the website: surveybasket.herokuapp.com
